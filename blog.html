Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM using Javascript. Here are some of the methods: getElementById(), getElementsByTagName(), getElementsByclassName(), getElementsByName(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML : This property represents the HTML content inside an element as a string, including all HTML tags. It is used when need to retrive or set both text and HTML content.

innerText : This property returns or sets the "rendered" text content of an element, excluding any HTML tags. It is used when need to retrieve or set visible text in an element but ignore any HTML tags or hidden content.

textContent : This property returns or sets the text content of an element, including all the texts in its child nodes but without any HTML tags. Its used when need to get or set the raw text content and ignore styling or hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in Javascript for handling events in a more efficient and flexible way by taking advantage of the event bubbling mechanism in the DOM.
Event delegation works by event bubbling, event listener on parent & determining the target.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that is triggered on an element will "bubble up" through its ancestors in the DOM tree, from the target element to the root of the document.
This process allows event listeners on parent elements to also catch and handle events triggered on child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element : To create new HTML elements, need to use document.createElement()

Adding an Element : to add the newly created element to the DOM, need to append it to an existing element. Append using parentElement.appendChild()

Removing an Element : to remove an element from the DOM, need to use removeChild() or remove()

Adding Text or HTML content : need to use innerText or innerHTML, to add text or HTML content to an element. innerText adds plain text. On the otherhand, innerHTML adds HTML content.
